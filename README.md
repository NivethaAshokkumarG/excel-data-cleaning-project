# excel-data-cleaning-project
Excel project demonstrating data cleaning, imputation, and analysis using formulas.
## Overview
This project demonstrates my hands-on skills in Excel for data cleaning and preprocessing.

## Key Skills Used
- Data Cleaning
- Handling Missing Values (ISBLANK, IF)
- Data Imputation (AVERAGEIF, AVERAGEIFS)
- Data Formatting
- Logical Functions

## File Details
- Excel file contains 4 sheets:
  1. Raw Data
  2. Cleaned Data
  3. Imputation Logic
  4. Final Output

To showcase my practical Excel skills for Data Analyst role.
